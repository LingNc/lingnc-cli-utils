# CLI工具库

## 工具列表
1. **easy-tun-proxy**: 在终端环境下进行TUN代理socks5流量，可以控制开关切换并自动配置路由表。
2. **multi-user-space**: 自动化配置Linux多用户开发环境，快速自动化创建新用户，并支持`brew`个人目录安装和`docker rootless`以及共享目录快速配置，提供用户私有控制工具。
3. **npc-install**: 对于Linux下的npc服务自动安装到系统服务（systemd），便于管理。
4. **drcom-install**: 对于dogcom客户端提供 Linux(systemd) / Windows(计划任务) 的一键安装、配置渲染与后台自启。
5. **miui-backup-edit**:
   1. **balatro_save**: 借助小米备份和其他工具实现对小丑牌存档的解析/读取/转移/归档操作，支持通过ADB无缝迁移，电脑/手机存档流转，电脑/手机存档的归档备份存储等功能。
   2. **miui-backup**: 解析小米备份，可以解析和还原备份之后的文件，进而在无root权限下保存/修改/恢复/转移应用数据。
